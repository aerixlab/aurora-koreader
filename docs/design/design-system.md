# 1. Layout Philosophy

Aurora prioritizes breathing room over density.

Every screen should feel organized, spacious and comfortable.

Readers should never feel overwhelmed by visual clutter.

## Reading Flow

 Top
  ↓
Primary Content
  ↓
Secondary Content
  ↓
Optional Actions

Example:

Screen
16px margin
  ↓
Section
24px gap
  ↓
Card
16px padding
  ↓
Text
12px gap
  ↓
Button

## Spacing System

XS = 4
S = 8
M = 12
L = 16
XL = 24
XXL = 32

## Grid

Home
1 column

Library
2-column grid (Book Covers)

Settings
1 column

Statistics
Cards stacked vertically

## Alignment

Everything aligns to the same invisible grid.

← Start
↓
Section Title
↓
Book Card
↓
Book Card
↓
Book Card

No random centering.
Except when intentionally used.

## Corner Radius
- Rounded

Small
6px

Cards
10px

Dialogs
14px

Buttons
8px

## Shadows
None. Instead use spacing.

## Dividers
Avoid unnecessary borders.
Prefer whitespace.
Only use dividers when needed.

## Density
Comfortable not compact not wasting space

Support 2 level

- Cozy
More spacing
Larger cards
Relaxed layout

- Compact
More books
Less spacing
Power users
