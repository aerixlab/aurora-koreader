# Current eReader Problems

## Too Many Clicks

Many e-readers require multiple taps to perform common actions.

Aurora Goal

Reduce the number of taps for frequently used actions.

---

## Hidden Features

Important features are difficult to discover.

Aurora Goal

Surface frequently used features while keeping advanced options available.

---

## Overwhelming Settings

Everything is displayed at once.

Aurora Goal

Separate common settings from advanced settings.

---

## Inconsistent Navigation

Different screens behave differently.

Aurora Goal

Use one consistent navigation model throughout Aurora.
